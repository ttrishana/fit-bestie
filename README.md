# fit-bestie
fitness app to help noobs at the gym (like me) structure their routine

# mvp scope

- pnboarding survey
- split selection
- exercise database
- simple exercise demos
- daily calorie and protein recommendation
- workout builder per day 
- active workout screen 
- home screen showing today's workout and macro targets
- progressive overload suggestions 
- basic user auth 

# stack

frontend: next.js, typescript, tailwind css, tanstack query, zustand, react hook form + zod
backend: next.js, prisma orm, bcryptjs, jsonwebtoken, zod
database: postgresql on supabase
auth: custom jwt
testing: pnpm, eslint + prettier, vitest, prisma studio
